# Notification Sounds

This directory is reserved for notification sound files.

## Planned Sounds

- `complete.mp3` or `complete.wav` - Sound when timer completes
- `tick.mp3` - Optional ticking sound during timer (future feature)

## Sound Requirements

- Format: MP3 or WAV
- Duration: 1-3 seconds recommended
- Volume: Moderate, not too loud
- Style: Pleasant, not jarring (e.g., bell, chime, gentle beep)

## Sources for Sounds

- [Freesound.org](https://freesound.org/) - Free sound effects
- [Zapsplat.com](https://www.zapsplat.com/) - Free sound effects
- Create your own using audio editing software
- Record your own sounds

## Note

Sound playback is currently handled by Chrome's notification system. Custom sound files can be added in a future update.
