# Icons

Please add the following icon files to this directory:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Creating Icons

You can:
1. Design your own icons using tools like Figma, Adobe Illustrator, or Canva
2. Use free icon generators online
3. Use emoji-based icons
4. Hire a designer

- Use a tomato (üçÖ) as the main symbol (representing Pomodoro)
- Use a clean, modern design with white background

### Quick Temporary Solution

For testing purposes, you can use emoji-to-icon converters or screenshot emoji and resize them to the required dimensions.
